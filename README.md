# BioTrace – Laboratory Information Management System (LIMS) for UNSRAT Biomolecular Lab

## Repository Structure
- backend/  → Laravel 11 API (ISO/IEC 17025 compliant modules)
- frontend/ → React SPA (Lab Dashboard & Sample Workflow)
- docs/     → Design docs, SRD, Traceability Table, etc.

## Branch Policy
- main → stable release branch
- dev  → active development branch

## Commit Convention
Use prefix:
- feat:   for new features
- fix:    for bug fixes
- docs:   for documentation
- refactor: for structure/cleanup
- test:   for unit/integration test commits
